# SaveTube

Простой загрузчик видео в очередь для macOS. Вставляешь ссылки → собираешь очередь → жмёшь **Скачать всё** → программа качает по одной. Под капотом [yt-dlp](https://github.com/yt-dlp/yt-dlp), работает с YouTube и ~1800 других сайтов.

## Установка (одна команда)

Открой **Терминал** (Cmd+Space → «Terminal») и вставь:

```bash
curl -fsSL https://raw.githubusercontent.com/holdem-lab/savetube/main/install.sh | bash
```

Скрипт поставит зависимости (`yt-dlp`, `ffmpeg`, `python-tk`) через Homebrew, скачает программу в `~/.savetube` и положит ярлык **SaveTube.command** на Рабочий стол.

> Нет Homebrew? Скрипт подскажет команду установки. Поставь и запусти снова.

## Запуск

Двойной клик по **SaveTube.command** на Рабочем столе.

Первый раз macOS может ругаться («неизвестный разработчик») — правый клик по файлу → **Открыть** → **Открыть**. Дальше открывается обычным кликом.

## Как пользоваться

1. Вставь ссылки в верхнее поле (по одной на строку) → **＋ Добавить в очередь**.
2. Выбери формат: **Видео mp4** или **Звук mp3**.
3. Кнопкой **Папка…** выбери, куда сохранять (запоминается).
4. **▶ Скачать всё** — качает по очереди.

Битая ссылка (приватное/удалённое видео) помечается ❌ и пропускается — очередь не встаёт.

## Обновление

Перезапусти команду установки — она перетянет свежую версию.
